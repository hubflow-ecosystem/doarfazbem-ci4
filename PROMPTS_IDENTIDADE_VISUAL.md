## Dicas de Uso

1. **Consistência**: Use sempre o mesmo estilo de personagem (stick figures com cabeças redondas)
2. **Cores**: Mantenha o emerald green (#10b981) como cor principal em todas as ilustrações
3. **Formato**: Sempre peça PNG com transparência para facilitar integração
4. **Tamanho**: Para web, gere em alta resolução (mínimo 1200px de largura)
5. **Variações**: Se uma imagem não ficar boa, peça "variações" mantendo o mesmo prompt
